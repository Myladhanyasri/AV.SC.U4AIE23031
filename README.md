# Campus Notification System

## Tech Stack

### Frontend
- React
- Material UI
- Axios

### Backend
- Node.js
- Express.js

---

## Features

- Notification Dashboard
- Backend API
- Logging Middleware
- REST API Integration
- Notification Cards
- Real-time architecture design
- Scalable system design documentation

---

## Run Frontend

```bash
cd notification_app_fe
npm install
npm run dev
```

---

## Run Backend

```bash
cd notification_app_be
npm install
node server.js
```
